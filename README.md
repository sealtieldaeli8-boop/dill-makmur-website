## 🔧 Maintenance

### Mengganti Gambar
Ganti URL placeholder gambar dengan URL gambar asli:
```html
<!-- Dari -->
<img src="https://via.placeholder.com/500x400?text=Bengkel+Motor">

<!-- Ke -->
<img src="path/to/your/image.jpg">
```

### Mengubah Warna
Edit warna di `css/style.css`:
```css
/* Warna primer */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Menambah Layanan
Tambahkan card baru di section layanan:
```html
<div class="col-md-6 col-lg-4 mb-4">
    <div class="card h-100 shadow-sm">
        <div class="card-body text-center">
            <i class="fas fa-icon fa-3x text-danger mb-3"></i>
            <h5 class="card-title">Nama Layanan</h5>
            <p class="card-text">Deskripsi layanan...</p>
        </div>
    </div>
</div>
```
