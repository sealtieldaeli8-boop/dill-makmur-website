# DILL MAKMUR - Bengkel Motor Terpercaya di Batam

Website profesional untuk bengkel motor DILL MAKMUR di Batam dengan desain modern, responsif, dan SEO-friendly.

## 🚀 Fitur Utama

- ✅ **Desain Responsif** - Sempurna di semua ukuran layar (mobile, tablet, desktop)
- ✅ **Galeri Profesional** - Showcase pekerjaan dengan overlay interaktif
- ✅ **Animasi Smooth** - Menggunakan AOS (Animate On Scroll)
- ✅ **Form Kontak** - Validasi form dan pengiriman pesan
- ✅ **SEO Optimized** - Meta tags dan struktur HTML yang optimal
- ✅ **Aksesibilitas** - Support untuk screen readers dan keyboard navigation

## 📁 Struktur File

```
dill-makmur-website/
├── index.html           # File HTML utama
├── style.css           # File CSS styling
├── script.js           # File JavaScript
└── docs/README.md      # Dokumentasi
```

## 🛠️ Teknologi yang Digunakan

- **HTML5** - Markup semantik
- **CSS3** - Styling dan animasi
- **Bootstrap 5** - Framework responsive
- **Font Awesome 6** - Icon library
- **AOS** - Animate On Scroll library
- **Vanilla JavaScript** - Interaktivitas
